플립 4주차 리눅스

명령어:
cat 파일이름 : 텍스트 파일내용확인
cat -n : 행번호까지 출력
ex) cat file1, cat -n file1

more 파일이름 : 파일내용이 엄청길어서 출력되지않은부분도 출력
ex) more file1
// space bar : 다음화면출력, enter : 한 줄씩 스크롤, q : 종료

less 파일이름 : 스크롤되어 지난 내용 확인
ex) less file1
// j : 한 줄씩 다음행, k : 한 줄씩 이전 행, space bar : 다음화면, ctrl+b : 이전화면, q : 종료

tail -5 파일이름 : 파일 뒷부분 5행 출력 //5대신 다른 숫자도 가능
ex) tail -7 /etc/services

cp 파일이름1 파일이름2 : 파일이름1을 파일이름2에 복사 //디렉토리도 가능
ex) cd file1 file2
옵션 : -i : file2가 존재하면 덮어쓸것인지, -r : 디렉토리를 복사할때 사용
ex) cd -i file1 file2
//첫번째인자가 파일이고 두번째인자가 디렉토리인 경우 두번째디렉토리안에 복사한 파일을 저장
//마지막 인자가 디렉토리인경우 앞에 파일이름으로 인자를 여러개 쓸수 있는데 이 경우 인자들의 파일이 모두 디렉토리안에 복사됨

mv 파일이름1 파일이름2 : cp와 같은 방식인데 복사가 아니라 이동하기, 이름을 바꾸는 기능도 수행
//두번째 인자가 디렉토리인경우, 해당디렉토리로 이동
//인자를 모두 디렉토리로도 사용가능(두번째가 존재하던 디렉토리면 그 아래로이동)
옵션: -i

rm 파일이름 : 삭제
옵션 : -i

rmdir 디렉토리이름 : 디렉토리삭제
옵션 : -r 하위 디렉토리 모두 삭제

ln 원본파일 링크파일 : 원본파일이 수정되면 링크파일도 원본파일과 똑같이 바뀌는 하드링크파일을 생성
