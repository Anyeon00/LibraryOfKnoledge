리눅스 플립3주차 플립

명령의 구조
형식: 명령 [옵션] [인자] // 옵션은 -나 --로 시작, 인자는 전달값
ex) ls -l // ls -l -a 또는 ls -la이렇게 옵션 두가지를 한번에 명령 가능
//ls --all 은 ls -a 와 같음 즉, -a라는 명령어를 풀어서 쓴것
// ls -all 이라고 쓰면 옵션 a, l, l 을 명령한것으로 해석함

파일
-일반파일(텍스트파일, 실행파일, 이미지파일)
-디렉터리 파일
-심벌릭 링크
-장치파일(하드디스크, 키보드 등)
*파일의 종류를 확인하는 명령
:file 파일이름


최상위 디렉토리 : / (트리구조이기 때문에 루트 디렉토리라고도 함)
서브딜렉토리(하위) 
부모 디렉토리(상위) : _ 로 표시


홈 디렉토리: 각 사용자에게 할당된 최상위 디렉토리(로그인시)(~로 표시)
작업 디렉토리: 현재 사용중인 디렉토리(. 으로 표시)
(현재 작업디렉토리위치: pwd로 확인)

 
절대경로명: 루트(/)디렉토리부터 시작(반드시 /로 시작)
상대경로명: 현재 디렉토리를 기준으로 시작
(상위폴더의 다른폴더에 있다면 .. 로 표시)
ex) ../~


배시셸 명령어
ls : 현재 디렉토리의 폴더들 표시		V
ls 파일이름(절대 또는 상대경로명으로 표시) : 해당파일의 폴더들 표시
(파일이름에 /로 시작하여 하나하나 다 써서 절대경로명으로도 쓸수잇음)
ls -F: 서브디렉토리의 파일종류표시 (/는 디렉토리, @는 심벌릭링크)
ls -a : 모든 파일 표시(숨겨진 파일도)
ls -d 파일이름: 그 디렉토리 자체에 대한 정보
clear : 화면 정리하는거
file 파일이름 : 파일종류 확인
pwd: 현재 위치한 디렉터리			V
mkdir week3 : 디렉토리 week3 만들기			V
mkdir sub1/subsub : sub1 아래에 subsub 만들기
mkrdir -p sub1/s1/s2/s3 : sub1부터 s3까지 한번에 만들기
touch a : 파일 a 만들기		V
rmdir 상대/절대경로명: 삭제
ex) ../../user 상위 상위에 있는 user라는 파일 삭제
cd week3 : week3로 들어가기(디렉토리 이동)		V
cd .. : 상위 디렉토리로 이동		V
*변수 만드는법
ex)
PS1 = "[\w]/$ "
