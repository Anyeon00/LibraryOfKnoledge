플립 5주차 리눅스
입출력 리다이렉트

명령어:
출력 리다이렉션
명령 > 파일명 : 덮어쓰기(기존 파일의 내용을 삭제하고 새로 결과를 저장)
명령 1> 파일명 : '' ( 0 이 입력, 1이 출력, 2가 오류인데 1이 출력이므로)
ex) ls -al > out1
ex) date > out1
* 지정한 이름의 파일이 없으면 새로 생성하여 수행함	
* set -o no clobber 이라고 쳐주고 >리다이렉션을 실행하면 파일내용이 겹쳐쓰이는 상황을 방지해줌
여기서 -o를 +o로 바꿔서 쳐주면 설정이 해제됨

>> 파일명 : 기존 파일의 내용 뒤(다음줄)에 결과를 추가
ex) date >> out1


오류 리다이렉션
명령 2> 파일명 : 표준 오류 메시지를 파일에 저장
ex) ls /abc 2> ls.err
*오류 메시지 버리기
ex) ls /abc 2> /dev/null
(/dev에 하드웨어들이 파일로 저장되어있는데 null이라는 가상의 파일도 존재함
 그래서 일로 보내면 다 사라짐)

*표준 출력과 표준 오류를 한번에 리다이렉션하기
ex) ls . /abc > ls.out 2> ls.err
(출력은 ls.out으로, 오류는 ls.err 로 보내라)
*출력과 오류 모두 같은파일에 리다이렉션하려면
ex) ls. /abc > ls.out 2>&1	


입력 리다이렉션 (직접 입력하지 않고 파일에 저장해놓은 내용을 입력으로 하고 싶을떄)
명령 < 파일명 : 표준 입력을 바꾼다
명령 0< 파일명 : ''
ex) mail syseong@bufs.ac.kr < ls.out






*모든 디렉토리를 보여주는 ls -al 시 나오는 .과 ..은 현재파일과 상위폴더를 가리킴,
 자동으로 만들어져 있음
